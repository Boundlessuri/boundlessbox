## 核心知识速查

### 化学平衡——常见平衡标志判断

| 条件 | 是否可作平衡标志 |
|------|----------------|
| 各组分浓度不变 | ✅ |
| 正逆反应速率相等（同一物质） | ✅ |
| 混合气体颜色不变 | ✅（有色气体参与时） |
| 混合气体密度不变（恒容） | ❌（密度=总质量/体积，总质量通常不变） |
| 加催化剂后速率不变 | ❌（催化剂不影响平衡位置） |

### 氧化还原——口诀

> 升失氧（化合价升高 → 失电子 → 被氧化 → 是还原剂）
> 降得还（化合价降低 → 得电子 → 被还原 → 是氧化剂）

### 离子方程式——不能拆成离子的情况

- 弱酸：HF、H₂S、H₂CO₃、CH₃COOH
- 弱碱：NH₃·H₂O、Fe(OH)₃
- 难溶物：BaSO₄、AgCl、CaCO₃
- 气体：CO₂、SO₂、H₂S
- 微溶物作反应物且为浊液：Ca(OH)₂（石灰乳）不拆，澄清石灰水才拆

### 原电池——锌铜稀H₂SO₄

```
负极（Zn，活泼）：Zn - 2e⁻ → Zn²⁺        氧化反应
正极（Cu，不活泼）：2H⁺ + 2e⁻ → H₂↑      还原反应
```

### 钠的重要反应

```
2Na + 2H₂O → 2NaOH + H₂↑
2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑
2NaHCO₃ --(加热)--> Na₂CO₃ + H₂O + CO₂↑
```

### 硫的重要反应

```
2H₂S + O₂(不足) → 2S + 2H₂O
2H₂S + 3O₂(足量) → 2SO₂ + 2H₂O
2H₂S + SO₂ → 3S + 2H₂O
Cu + 2H₂SO₄(浓) --(加热)--> CuSO₄ + SO₂↑ + 2H₂O
```
